# 5분 후 알람 봇

사용자가 읽고 싶은 블로그 글의 링크를 입력하면 제목을 자동으로 크롤링하고,
5분 뒤 팝업 알람으로 읽기 시간을 알려주는 Tkinter 기반 데스크톱 애플리케이션입니다.

## 주요 기능

- URL 입력 후 제목 자동 크롤링 (BeautifulSoup 사용)
- 등록된 제목을 최신 순으로 리스트에 표시
- 링크마다 5분 타이머를 개별적으로 실행하고 완료 시 팝업 알림 표시
- 프로그램 종료 시 알람 타이머 자동 정리

## 실행 방법

1. Python 3.9 이상이 설치되어 있어야 합니다.
2. 필요한 패키지를 설치합니다.

   ```bash
   pip install -r requirements.txt
   ```

3. 애플리케이션을 실행합니다.

   ```bash
   python alarm_bot.py
   ```

4. 실행된 창에서 블로그 URL을 입력하고 **등록** 버튼을 누르면 5분 후 알람 팝업이 표시됩니다.

> **참고:** 일부 사이트는 봇 접근을 차단하거나 제목 태그가 표준과 다를 수 있으므로 제목을 불러오지 못할 수 있습니다.
